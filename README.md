# [FM] FreeMAX

УСТАНОВКА И НАСТРОЙКА
=====================
1) Установите Python 3.10 или новее.
sudo apt install python3
sudo apt install python3-pip

3) Установите зависимости:
pip install -r requirements.txt

4) Установите браузеры Playwright:
playwright install

ПОДГОТОВКА БОТА
===============
1) Создайте Telegram-бота через @BotFather и получите токен.

2) Создайте файл token.txt и вставьте в него токен бота.

3) Зашифруйте токен:
python3 enctok.py

После этого появится файл token.enc.
Файл token.txt можно и нужно удалить.

РЕГИСТРАЦИЯ ПОЛЬЗОВАТЕЛЕЙ
=========================
Перед использованием пользователь должен:

1) Авторизовать браузер на сервере через:
python3 new_user.py

2) Ввести кодовое имя сессии и пароль (кодовое имя можно ввести за него, проследите за правильностью айди).

После этого будет создан файл:
sessions/имя_сессии.enc

Сессия хранится в зашифрованном виде.

ЗАПУСК БОТА
===========
python3 botR.py

При запуске:
- Введите пароль для расшифровки token.enc
- Бот уведомит пользователей, если был перезапуск

ОНЛАЙН-ЧТЕНИЕ
=============
- Можно читать только один чат одновременно.
- Если бот перезапустится - пользователю с включенным ранее чтением придёт уведомление.
- Для продолжения нужно:
1) Открыть сессию (ввести пароль)
2) Запустить онлайн-чтение заново

ФАЙЛЫ ПРОЕКТА
=============
bot.py - основной файл бота
new_user.py - регистрация новой сессии
open_session.py - открыть сессию для просмотра (headless=False)
enctok.py - шифрование токена
sessions/ - зашифрованные сессии пользователей
token.enc - зашифрованный токен бота
reading_users.json - список пользователей с активным онлайн-режимом

БЕЗОПАСНОСТЬ
============
- Сессии хранятся зашифрованными (Fernet + PBKDF2)
- Токен бота хранится только в зашифрованном виде
- Пароли нигде не сохраняются
- В случаем компрометации сервера у гада-проказника-бандита не будет возможности запустить копию бота под тем же ником, украсть сессии
- При внезапном/запланированном выключении сервера никакие чувствительные данные не останутся в открытом виде. Для их чтения придётся снова ввести пароль
